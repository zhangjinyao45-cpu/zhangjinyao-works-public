# 苏予的视觉规范库

## 推荐配色方案（按风格）

### 极简优雅
- 主色：#1A1A1A（近黑）
- 辅色：#666666（深灰）
- 强调：#FF4D4F（克制的红）
- 背景：#FAFAFA / #FFFFFF

### 科技理性
- 主色：#0066FF（深蓝）
- 辅色：#4A5FFF / #00C2FF
- 背景：#F5F7FA
- 适合：B 端 / 工具类 / 数据类产品

### 温暖亲切
- 主色：#FF8C42（橙）
- 辅色：#FFC857
- 背景：#FFF8F0
- 适合：餐饮 / 生活方式 / 家庭类产品

### 商务严肃
- 主色：#003366（深蓝黑）
- 辅色：#666666
- 强调：#C49A6C（金）
- 适合：金融 / 法务 / 高端服务

## 字体推荐

### 中文
- **PingFang SC**：苹果系统默认，阅读友好（首选）
- **思源黑体（Source Han Sans）**：开源、可商用
- **HarmonyOS Sans**：华为系，现代感

### 英文
- **Inter**：开源、专为屏幕设计（首选）
- **SF Pro**：苹果系统
- **Roboto**：Google 系

### 字体组合（中英混排）
- PingFang SC + Inter（最稳）
- 思源黑体 + SF Pro
- HarmonyOS Sans + HarmonyOS Sans Latin

## 字号阶梯（6 级）

```
H1 标题: 32px
H2 副标题: 24px
H3 章节: 20px
正文加粗: 16px
正文: 14px
辅助文字: 12px
```

## 间距规范

```
4px  — 紧密元素（图标 + 文字）
8px  — 同组元素之间
12px — 同模块的次要分隔
16px — 同模块的主要分隔
24px — 不同模块之间
32px — 大模块之间
48px — 屏幕分区之间
```

## 圆角规范（统一选一种）

```
风格1：锐利（全 0px）— 商务严肃
风格2：现代（全 4-6px）— 大多数产品
风格3：友好（全 8-12px）— 消费类
风格4：可爱（全 16-24px）— Z 世代
```

## 阴影规范

```
轻量浮起：0 1px 4px rgba(0,0,0,0.08)
卡片浮起：0 2px 8px rgba(0,0,0,0.10)
弹窗浮起：0 8px 24px rgba(0,0,0,0.16)
```

## Pexels 图片选择标准

### 优先选
- 空间氛围图（窗光、自然光）
- 真实场景（不是摆拍）
- 留白充足（适合做背景或叠字）
- 风格情绪明确（暖光 / 冷调 / 复古 / 极简）

### 避免选
- 商业广告感（人物对镜头摆 pose）
- 廉价网红风（过度滤镜）
- 低分辨率（< 1920px 宽）
- 同页面风格冲突

### 搜索词必须用英文 + 场景化
- ✅ `moody coffee shop interior`
- ✅ `editorial restaurant interior`
- ❌ `咖啡馆`
- ❌ `好看的图`

## 一致性铁律

**同一个产品里，以上所有规范一旦选定，全产品统一，不允许"这里 8px 那里 12px"。**
