# 阶段3：线框图与交互规范

## 目标
基于 PRD，生成详细的线框图和交互流程规范，让阶段4 的原型开发有完整的实现依据。

## 输入

- `02-产品需求文档.md`

## 线框图必须包含

1. **完整的页面结构和所有区块**
   - 页面级线框（每个页面的整体布局）
   - 区块级线框（每个区块的内容和样式）

2. **详细的交互流程**
   - 点击 A → 显示 B
   - 输入 X → 触发 Y
   - 状态切换路径

3. **所有 UI 组件及其状态**
   - 按钮：默认 / hover / active / disabled / loading
   - 表单：默认 / 输入中 / 错误 / 成功
   - 列表：默认 / 空 / 加载中 / 错误

4. **导航逻辑和转场动画**
   - 顶部导航 / 底部导航 / 侧边导航
   - 页面切换动画（滑入 / 渐变 / 弹出）

5. **组件间的数据流**
   - 哪些组件读取共享状态
   - 哪些组件触发状态变更
   - 数据更新如何传播
