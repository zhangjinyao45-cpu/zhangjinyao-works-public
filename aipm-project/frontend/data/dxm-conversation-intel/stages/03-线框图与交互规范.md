## 五、交互规范

### 5.1 加载/空/错三态
| 状态 | 视觉 | 文案 |
|---|---|---|
| Loading | 骨架屏占位（与最终内容形态一致） | （无） |
| Empty | 居中插图 + 文案 + 重置按钮 | "暂无符合条件的对话" |
| Error | 顶部红条 + 错误码 + 重试按钮 | "数据加载失败，请重试" |

### 5.2 反馈
- **Toast**：右下 / 8s 自消失 / 成功绿 / 警告黄 / 错误红
- **Modal**：居中 600px / 半透明遮罩 / ESC 关闭
- **审核确认**：「驳回」需输入理由（≥10 字才可提交）

### 5.3 键盘快捷键
| 快捷键 | 动作 |
|---|---|
| `J` / `K` | 上下条切换 |
| `L` | 标记已读 |
| `R` | 触发重新分类 |
| `P` | 跳到 Prompt 中心 |
| `/` | 全局搜索聚焦 |
| `ESC` | 返回上一层 |

### 5.4 高净值视觉强化
- 列表行：左侧 3px 金色竖条 (▌)
- 详情/画像：用户名右侧 [VIP] 金徽章
- Prompt 编辑：顶部「VIP 客户」金色标签
