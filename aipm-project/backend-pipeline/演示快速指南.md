# 🎭 AIPM 演示模式 · 快速使用指南

## 现在就能用

### 第 1 步：开启演示模式
双击：
```
C:\Users\zhangjinyao01_dxm\aipm-project\backend-pipeline\demo-on.bat
```

### 第 2 步：打开网站
```
http://localhost:3000/app/start.html
```

### 第 3 步：演示
- 在 start.html 输入**任意项目名 + 任意想法**
- 点"开始流水线"
- 网站自动跳转到 stage0.html，**周明开始按度小满项目剧本聊天**
- 用户随便回答 7 轮（内容不影响，剧本固定）
- 点"生成需求文档"，跳到剧场页 theatre.html
- **五人评审会自动流式播放**（10 条发言 + 3 个插话）
- 跳到 decisions.html 拍板 3 个决策（任意选）
- 进 Stage 0.5 顾青竞品分析，再评审、再决策（2 个）
- 进 Stage 01 / 02 / 03 / 04（每阶段都有真实的产物 + 评审会重放）
- 最终看到 Stage 04 高保真原型

整个流程**与真实 AIPM 完全一致**——但内容固定为度小满那个高质量演示版本。

### 第 4 步：演示完关闭
双击：
```
C:\Users\zhangjinyao01_dxm\aipm-project\backend-pipeline\demo-off.bat
```

关闭后**新创建的项目走真实 Claude API**，但已经创建的演示项目会继续按演示行为运行（避免半路切换坏掉）。

## 重要文件清单

```
backend-pipeline/
├── demo-on.bat              ← 双击开启
├── demo-off.bat             ← 双击关闭
├── DEMO-MODE-README.md      ← 详细技术说明
├── .demo-mode               ← 标记文件（开关）
└── src/services/
    └── demo-mode.js         ← 演示模式核心逻辑
```

## 网站界面

完全没有任何"演示模式"标识——和真实模式视觉、交互完全一致。

## 切换原理

后端**每次请求实时检查** `.demo-mode` 文件是否存在：
- 存在 → 演示模式
- 不存在 → 真实模式

不需要重启服务器，双击 bat 立即生效。

## 如果出问题怎么办

### 后端没启动
```powershell
$ownerIds = Get-NetTCPConnection -LocalPort 3000 -ErrorAction SilentlyContinue | Select-Object -ExpandProperty OwningProcess -Unique
foreach ($ownerId in $ownerIds) { Stop-Process -Id $ownerId -Force }
Start-Process -FilePath node -ArgumentList 'src/server.js' -WorkingDirectory 'C:\Users\zhangjinyao01_dxm\aipm-project\backend-pipeline' -WindowStyle Hidden
```

### 演示项目堆积太多
```powershell
cd C:\Users\zhangjinyao01_dxm\aipm-project\workspace\projects
Get-ChildItem -Directory | Where-Object {
  $f = "$($_.FullName)\meta.json"
  if (Test-Path $f) {
    $meta = Get-Content $f -Raw | ConvertFrom-Json
    return ($meta._demoMode -eq $true) -and ($meta.projectId -ne 'dxm-conversation-intel')
  }
  return $false
} | Remove-Item -Recurse -Force
```

### 源项目不小心被改坏
```powershell
# 从 frontend/data 备份恢复
cd C:\Users\zhangjinyao01_dxm\aipm-project
Remove-Item workspace\projects\dxm-conversation-intel -Recurse -Force
Copy-Item frontend\data\dxm-conversation-intel workspace\projects\dxm-conversation-intel -Recurse
```
