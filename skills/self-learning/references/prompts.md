# 抽取与摘要提示词模板

## 模板 A：从会话日志抽取候选能力

```
你正在执行「自我学习」的抽取阶段。请从下面的会话内容中识别出**可复用的能力模式**，
而不是一次性的具体任务。

输出 JSON：
{
  "candidate_capabilities": ["<≤30字>", ...],
  "lessons": ["<踩坑或反例>", ...],
  "user_preferences": ["<用户偏好>", ...]
}

判定标准：
- 可复用 = 换一个用户、换一份数据，依然能套用。
- 不可复用 = 仅在当前业务/数据/人物下成立。

会话内容：
<<<
{conversation}
>>>
```

## 模板 B：从文档抽取领域知识

```
你正在为「原力AI助手」补充领域知识。请从下面的文档中抽取：
1. 关键概念与术语（≤10个）
2. 标准操作流程（如有）
3. 常见错误与规避方式

输出 Markdown，分三段。

文档内容：
<<<
{document}
>>>
```

## 模板 C：冲突消解

```
现有能力条目：
A. <旧>
B. <新>

请判断：
- 是否描述同一能力？
- 哪一条更具体、更可执行？
- 是否需要保留两条（适用场景不同）？

输出：keep_a / keep_b / keep_both / merge:<合并后描述>
```
